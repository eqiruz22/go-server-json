
# FAKE JSON REST API

This is a simple REST API WITH JSON DATABASE
inspired by JSONPLACHOLDER


## Installation

Install go-server-json

```bash
  git clone git@github.com:eqiruz22/go-server-json.git
  cd go-server-json
```
    
## Running Tests

To run tests, run the following command

```bash
  go run server.go
```


## Features

- GET 
- CREATE 
- GET ID 
- PATCH 
- DELETE
